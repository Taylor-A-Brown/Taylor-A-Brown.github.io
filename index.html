Latest Project

Word-Masters

Word Masters is a Wordle-inspired word guessing game built with HTML, CSS, and JavaScript. Players have six attempts to solve a five-letter word puzzle using color-coded feedback to guide their guesses. This project demonstrates DOM manipulation, API integration, asynchronous JavaScript, and interactive game development.

Latest Project

Tenzies

Tenzies is a dice game built with React and Vite where players roll until all ten dice match the same value. Players can hold individual dice between rolls, with dynamic game state updates and win detection. This project demonstrates React state management, component-based design, hooks, accessibility improvements, and interactive UI development.
